# Motion Detector Using Arduino UNO

## Overview
A motion detection system developed using Arduino UNO and a PIR sensor to detect human movement and trigger alerts.

## Features
- Motion detection using PIR sensor
- LED indication
- Buzzer alert
- Real-time response

## Technologies Used
- Arduino UNO
- PIR Sensor
- Arduino IDE
- C/C++
